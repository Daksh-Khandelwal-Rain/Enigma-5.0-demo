# ENIGMA 5.0

ENIGMA 5.0 is an interactive static event website for the SIES CSI hackathon. It combines a transmission intro, mechanical plugboard, animated dossiers, decryption effects, timeline, registration drawer, and ambient particle field.

## Technologies

- Semantic HTML
- Custom CSS and Tailwind CSS (CDN)
- Browser-native JavaScript

## Structure

```text
index.html             Page markup and external resource links
assets/css/styles.css  All custom presentation rules
assets/js/main.js      Tailwind setup and the complete interactive runtime
```

## Run

Open `index.html` in a modern browser. For development, serve this directory with any simple static HTTP server.
