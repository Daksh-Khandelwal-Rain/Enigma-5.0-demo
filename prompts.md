# AI Prompt Documentation

This project was developed with the assistance of AI tools (primarily ChatGPT and Claude). The AI was used to brainstorm ideas, improve the UI/UX, refactor code, and assist with responsive design. The overall concept, theme selection, design direction, testing, and final implementation decisions were made by me.

---

## Prompt 1 – Initial Concept

Design a landing page for "Enigma 5.0", the flagship hackathon of SIES CSI. The website should have a mysterious, encrypted communication theme inspired by the word "Enigma" rather than a generic hackathon landing page.

---

## Prompt 2 – UI Improvements

Improve the visual hierarchy, typography, spacing, animations, and user experience while preserving the Enigma theme.

---

## Prompt 3 – Responsive Design

Make the website fully responsive for desktop, tablet, and mobile devices without changing the visual identity.

---

## Prompt 4 – Code Refactoring

Separate the project into HTML, CSS, and JavaScript files while preserving all functionality, animations, and styling.

---

## Prompt 5 – Content Refinement

Improve headings, section descriptions, and call-to-action buttons while keeping the mysterious atmosphere consistent.
